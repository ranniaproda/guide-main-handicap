# guide-main-handicap

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存储，可直接通过浏览器访问，无需后端服务支持。

## 项目简介

本项目提供了一组独立的 HTML 页面，每个页面均包含完整的结构与样式，适用于各类参考、展示或轻量级工具场景。所有页面均为自行编写或整理，不依赖外部特定域名或网站。

## 目录说明

```
guide-main-handicap/
├── index.html          # 仓库主页（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── example-1.html  # 示例页面 1
│   ├── example-2.html  # 示例页面 2
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `pages/` 目录下为独立归档的 HTML 文件，每个文件可单独使用。
- `assets/` 目录存放页面共享的样式、脚本或图片资源，如需修改全局样式可在此操作。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，部分页面可能引用 `assets/` 下的公共资源。
- 页面内容可能涉及多种主题或功能，具体用途请参考文件名或页面内注释。
- 归档页面不会自动收集用户信息，不包含跟踪代码或外部请求（除非在页面内明确声明）。

## 维护说明

- 本仓库不定期更新，新增或修改页面会记录在提交日志中。
- 欢迎通过 Issue 提出建议或报告问题，但请注意不针对特定网站或域名。
- 如需贡献新页面，请确保内容独立、不侵犯第三方权益，并遵循现有目录结构。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接使用浏览器打开 `pages/` 目录下的任意 HTML 文件。
3. 或部署至任何静态文件托管服务（如 GitHub Pages、Netlify 等）后访问。
